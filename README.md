 Ashraf's Vision Medicine Store

A simple medicine store demo modeled after an online pharmacy interface.

## Features

- Search medicines by name or description
- Filter products by categories
- Add medicines to cart
- Client login and signup before checkout
- Delivery details collection before payment
- Selectable payment methods: Card, UPI, Wallet, and Cash on Delivery
- Razorpay payment flow integration for online methods
- Backend order submission and delivery partner assignment
- Real-time delivery tracking with ETA updates
- Teleconsultation booking and doctor video call preview
- Responsive partner confirmation UI

## Files

- `index.html` — App UI and page structure
- `styles.css` — Styling for dashboard, products, cart, and delivery screens
- `app.js` — Product filtering, cart logic, payment, and backend integration
- `server.js` — Backend server for order persistence and delivery partner assignment
- `package.json` — Backend dependencies and start script
